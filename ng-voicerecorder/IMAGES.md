# アイテム画像について

このボイスレコーダーリソースでは、以下の画像ファイルが必要です：

## 必要な画像ファイル

### 1. voice_recorder.png
- **説明**: ボイスレコーダーアイテムの画像
- **推奨サイズ**: 256x256px または 512x512px
- **ファイル形式**: PNG（透明背景推奨）

### 2. empty_tape.png
- **説明**: 空の録音テープアイテムの画像
- **推奨サイズ**: 256x256px または 512x512px
- **ファイル形式**: PNG（透明背景推奨）

### 3. recorded_tape.png
- **説明**: 録音済みテープアイテムの画像
- **推奨サイズ**: 256x256px または 512x512px
- **ファイル形式**: PNG（透明背景推奨）

## 画像の配置場所

**qb-coreの場合**:
```
qb-inventory/html/images/
├── voice_recorder.png
├── empty_tape.png
└── recorded_tape.png
```

**ox_inventoryの場合**:
```
ox_inventory/web/images/
├── voice_recorder.png
├── empty_tape.png
└── recorded_tape.png
```

## 画像デザインのガイドライン

### ボイスレコーダー (voice_recorder.png)
- 小型のデジタル録音機をイメージ
- カラー: ブラック、シルバー、または濃いグレー
- 表示画面やボタンがあると良い

### 空のテープ (empty_tape.png)
- カセットテープまたはminiDVテープのような外観
- カラー: ライトグレー、ホワイト、またはベージュ
- ラベル部分は空白または「BLANK」表示

### 録音済みテープ (recorded_tape.png)
- 空のテープと基本デザインは同じ
- カラー: より濃い色（黒、濃いグレー、青など）
- ラベル部分に録音済みを示すマークやテキスト

## 画像作成のヒント

1. **一貫性**: 3つの画像は同じスタイル・視点で統一する
2. **視認性**: インベントリで小さく表示されても識別しやすいデザイン
3. **透明背景**: PNG形式で背景を透明にする
4. **解像度**: 高解像度で作成し、必要に応じてリサイズ

## 無料リソース

アイコン作成に使える無料ツール・リソース：
- **Canva**: オンラインデザインツール
- **GIMP**: 無料の画像編集ソフト
- **Flaticon**: アイコン素材サイト
- **Icons8**: アイコン素材サイト
