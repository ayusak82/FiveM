# ng-sidejob

ミニゲームをクリアしておこずかいをもらう内職スクリプト

## 作者情報

- **作者**: NCCGr
- **問い合わせ先**: Discord: ayusak
- **バージョン**: 1.0.0

## 説明

プレイヤーが内職場所でミニゲームに挑戦し、成功すると報酬を獲得できるスクリプトです。
5種類のオリジナルミニゲームから選択でき、それぞれ異なる難易度と報酬が設定されています。

## 機能

### ミニゲーム一覧

1. **データ入力作業（タイピングゲーム）**
   - 難易度: 簡単
   - 報酬: $100 - $300
   - 表示された文字列を正確に入力

2. **部品検品作業（色合わせゲーム）**
   - 難易度: 普通
   - 報酬: $150 - $350
   - 指定された色のボックスを順番にクリック

3. **在庫確認作業（記憶ゲーム）**
   - 難易度: 普通
   - 報酬: $200 - $400
   - 表示されたアイテムを記憶して選択

4. **組み立て作業（リズムゲーム）**
   - 難易度: 普通
   - 報酬: $180 - $380
   - タイミングに合わせてスペースキーを押す

5. **梱包作業（パズルゲーム）**
   - 難易度: 難しい
   - 報酬: $250 - $500
   - ブロックを配置してラインを消す（テトリス風）

### その他の機能

- クールダウンシステム（デフォルト: 5分）
- スコアに応じた報酬変動
- マーカー表示とインタラクション
- ox_libを使用したメニューシステム

## 依存関係

- **qb-core** (必須)
- **ox_lib** (必須)

## インストール方法

1. `ng-sidejob` フォルダをサーバーの `resources` フォルダに配置
2. `server.cfg` に以下を追加:
   ```
   ensure ng-sidejob
   ```
3. サーバーを再起動

## 設定

`shared/config.lua` で以下の設定が可能です:

### 内職場所の設定
```lua
Config.JobLocations = {
    {
        coords = vector3(1274.5, -1710.5, 54.77),
        heading = 90.0,
        label = "内職受付"
    }
}
```

### ミニゲーム設定
各ミニゲームの制限時間、報酬範囲などをカスタマイズ可能

### クールダウン設定
```lua
Config.Cooldown = 300 -- 秒単位（デフォルト: 5分）
```

### デバッグモード
```lua
Config.Debug = false -- trueにするとコンソールにログが表示されます
```

## 使用方法

1. 内職場所（緑のマーカー）に近づく
2. `[E]` キーを押してメニューを開く
3. 好きなミニゲームを選択
4. ミニゲームをクリアして報酬を獲得
5. クールダウン後、再度挑戦可能

### ミニゲーム操作方法

#### タイピングゲーム
- キーボードで文字を入力

#### 色合わせゲーム
- マウスで指定された色のボックスをクリック

#### 記憶ゲーム
- アイテムを記憶後、マウスで選択

#### リズムゲーム
- `スペースキー` でタイミングを合わせる

#### パズルゲーム
- `←` `→` : 左右移動
- `↓` : 高速落下
- `↑` : 回転

#### 共通操作
- `ESC` : ゲームを中止

## トラブルシューティング

### ミニゲームが表示されない
- F8コンソールでエラーを確認
- 依存関係（qb-core、ox_lib）が正しく起動しているか確認

### 報酬が受け取れない
- サーバーコンソールでエラーを確認
- qb-coreが正しく動作しているか確認

### マーカーが表示されない
- `shared/config.lua` の座標が正しいか確認
- リソースが正しく起動しているか確認
