# ox_itemlist

シンプルなアイテムリスト表示スクリプトです。ox_inventoryのアイテム情報を使用して、画像付きのリストを表示します。

## 必要条件

- ox_lib
- ox_inventory

## インストール方法

1. このリソースを`resources`フォルダにコピーします。
2. `server.cfg`に以下の行を追加します：

```cfg
ensure ox_lib
ensure ox_inventory
ensure ox_itemlist
```

## 使用方法

以下の2つの方法でアイテムリストを表示できます：

1. チャットコマンド: `/items`
2. キーバインド: `F3`キー（デフォルト）

## 機能

- ox_inventoryに登録されているすべてのアイテムを表示
- アイテムの画像表示
- アイテムの詳細情報（説明、重量、スタック可否）
- アイテム選択時の通知表示

## カスタマイズ

キーバインドを変更する場合は、`client.lua`の`defaultKey`の値を変更してください。
