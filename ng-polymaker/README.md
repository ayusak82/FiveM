# ng-polymaker

FiveM用のPolyzoneエリア作成補助スクリプトです。
設定したポイントをvector2形式でクリップボードにコピーできます。

## 機能
- コマンドでポイント設定モードを開始
- 設定したポイントをvector2形式でクリップボードにコピー
- 直感的な操作とUI表示

## 必要要件
- qb-core
- ox_lib

## インストール方法
1. `ng-polymaker`フォルダをサーバーのresourcesフォルダに配置
2. server.cfgに以下を追加:
```cfg
ensure ng-polymaker
```

## 使用方法
1. ゲーム内で `/polymaker` コマンドを実行
2. 以下のキーで操作:
   - `E`: 現在の位置にポイントを追加
   - `X`: 最後に追加したポイントを削除
   - `ENTER`: 設定完了（クリップボードにコピー）
   - `ESC`: キャンセル
