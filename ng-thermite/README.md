# ng-thermite

テルミットミッションスクリプト - チームベースの強盗ミッション

## 作者情報
- **作者**: NCCGr
- **問い合わせ先**: Discord: ayusak

## 説明
プレイヤーがチームを組んで軍事基地から車両を盗み出し、バイヤーに届けるミッションスクリプトです。

## 主な機能
- チームシステム（2-6人）
- 警察人数チェック
- クールダウンシステム
- 軍事基地での警備員戦闘（25人のNPC）
- 車両盗難・配達ミッション
- 報酬配布（チーム全員に均等分配）
- Discord Webhook対応
- ox_target / qb-target 対応

## 依存関係
### 必須
- qb-core
- ox_lib
- qb-target または ox_target
- qb-phone（メール通知用）
- vehiclekeys（車両キーシステム）
- cdn-fuel（燃料システム）

### オプション
- okokNotify（通知システム）

## インストール方法
1. `ng-thermite` フォルダをサーバーの `resources` フォルダに配置
2. `server.cfg` に `ensure ng-thermite` を追加
3. `shared/config.lua` で設定をカスタマイズ
4. サーバーを再起動

## 設定
`shared/config.lua` で以下の設定が可能：
- クールダウン時間
- 必要な警察の人数
- チームの最小・最大人数
- 報酬金額
- 警備員の配置・強さ
- Discord Webhook URL
- ターゲットシステムの選択

## 使用方法
1. ミッション開始地点（デフォルト: -604.0787, -773.9486, 24.403778）に行く
2. NPCに話しかけてチームに参加
3. チームリーダーがミッションを開始
4. 軍事基地に向かい、警備員を倒す
5. 車両を盗んでバイヤーの場所まで運ぶ
6. 報酬を受け取る
