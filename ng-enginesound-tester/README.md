# ng-enginesound-tester

FiveMサーバーに導入されているカスタムエンジン音を簡単に変更・テストできるスクリプトです。
車両に乗車中、コマンド一つでエンジン音を変更できます。

## 作者情報
- **作者**: NCCGr
- **問い合わせ先**: Discord: ayusak

## 機能
- シンプルなコマンドでエンジン音を変更
- 一時的にエンジン音を適用（車両から降りると自動で元に戻る）
- 利用可能なエンジン音のリストを自動表示
- ox_libメニューUIで簡単操作
- カスタムエンジン音名の手動入力に対応
- デバッグモード搭載

## 依存関係
- **ox_lib** (必須)

## インストール方法

1. `ng-enginesound-tester` フォルダをサーバーの `resources` フォルダに配置
2. `server.cfg` に以下を追加：
```cfg
ensure ox_lib
ensure ng-enginesound-tester
```
3. サーバーを再起動

## 設定方法

`shared/config.lua` を編集して設定をカスタマイズできます：

### デバッグモード
```lua
Config.Debug = true  -- 開発時はtrue、本番環境ではfalse
```

### エンジン音リストのカスタマイズ
サーバーに導入されているカスタムエンジン音名を追加してください：
```lua
Config.CommonEngineSounds = {
    -- バニラGTA5エンジン音
    'SULTAN',
    'ELEGY',
    'COMET',
    -- ... (多数のバニラエンジン音が含まれています)
    
    -- カスタムエンジン音
    'v8engine',
    'v6engine',
    'v12engine',
    -- 追加のカスタムエンジン音名をここに記載
}
```

## 使用方法

### コマンド

#### メニューを開く
```
/engine
```
利用可能なエンジン音のリストが表示されます。選択してエンジン音を適用できます。

#### 直接エンジン音を適用
```
/engine [エンジン音名]
```
例：
- `/engine SULTAN` - スルタンのエンジン音を適用
- `/engine v8engine` - カスタムV8エンジン音を適用
- `/engine ELEGY` - エレジーのエンジン音を適用

#### エンジン音を元に戻す
```
/stop
```
元のエンジン音に戻します。

### 使用の流れ

1. 車両に乗車
2. `/engine` コマンドでメニューを開く
3. リストから好きなエンジン音を選択
4. エンジン音が即座に変更されます
5. `/stop` で元に戻すか、車両から降りると自動で元に戻ります

### 便利な使い方

#### 特定の車両のエンジン音を試す
```
/engine SULTAN
```
スルタンのエンジン音を現在の車両に適用

#### カスタムエンジン音を試す
```
/engine v12engine
```
カスタムV12エンジン音を適用

#### メニューから選択
```
/engine
```
メニューを開いて、リストから選択（検索も可能）

### 注意事項

- エンジン音は一時的な変更です（車両から降りると自動で元に戻ります）
- 同時に複数のエンジン音を適用することはできません
- 既にエンジン音が適用されている場合は、先に `/stop` で元に戻してください

## トラブルシューティング

### エンジン音が変わらない
- エンジン音名が正しいか確認してください（大文字小文字を区別します）
- サーバーにエンジン音が正しく導入されているか確認してください
- デバッグモードを有効にして、コンソールログを確認してください

### メニューにエンジン音が表示されない
- `shared/config.lua` の `Config.CommonEngineSounds` にエンジン音名を追加してください
- カスタム入力機能を使用して、手動でエンジン音名を入力することもできます

### エンジン音が元に戻らない
- `/stop` コマンドを実行してください
- 車両から降りて再度乗車してください
